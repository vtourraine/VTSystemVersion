# VTSystemVersion

_Checking the current iOS device system version._

Adapted from this [Stack Overflow answer](http://stackoverflow.com/a/5337804/135712).


## How to Use

``` objc
if ([UIDevice vt_systemVersionIsGreaterThan:@"9.0"]) {
  // ...
}
```
